# 🌤️ Weather App

A real-time Weather Application built as part of my Full Stack Developer portfolio.

## 🛠️ Tech Stack
- **Frontend:** React.js, JavaScript (ES6+), Bootstrap
- **API:** OpenWeatherMap REST API
- **Tools:** Axios, Git, GitHub

## ✨ Features
- Real-time weather data by location
- 5-day forecast display
- Async error handling and loading states
- Fully responsive Bootstrap UI

## 🚀 Getting Started
git clone https://github.com/yelchuri-chandrasekhar/weather-app
cd weather-app
npm install
npm start
